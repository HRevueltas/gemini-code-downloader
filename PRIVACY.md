# Privacy Policy for Code Exporter for Gemini

**Effective Date:** March 22, 2026

This Privacy Policy describes how "Code Exporter for Gemini" (the "Extension") handles your data. We are committed to protecting your privacy and ensuring a secure, fully local experience.

## 1. Data Collection

**We do not collect, store, transmit, or share any of your personal data, user data, or generated code.** The Extension operates entirely locally within your web browser. It does not contain any tracking software, telemetry, or third-party analytics (such as Google Analytics).

## 2. How the Extension Works

The Extension is designed with a single purpose: to add a download button to the code blocks on the Google Gemini web interface.

- When you click the download button, the Extension reads the text content of that specific code block directly from your browser's DOM (Document Object Model).
- It then generates a file locally on your device using built-in browser APIs.
- **None of the code, conversations, or prompts are ever sent to external servers or databases.**

## 3. Required Permissions

To function properly, the Extension requires the following permission:

- **Host Permission (`https://gemini.google.com/*`):** This permission is strictly used to inject the necessary user interface (the download button) and the local script into the Gemini interface. The Extension remains completely inactive on all other websites.

## 4. Third-Party Services

This Extension does not use any third-party services, APIs, or external code (remote code). Everything required for the Extension to run is bundled within the installation package.

## 5. Changes to this Privacy Policy

If we make any changes to this Privacy Policy, we will update this document and modify the "Effective Date" at the top. Since we do not collect user data or emails, we encourage users to review this page periodically.

## 6. Contact

If you have any questions or concerns about this Privacy Policy or the Extension's data practices, please open an issue on our GitHub repository.
